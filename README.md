# package-create

> Scaffolding tool for NPM package project

## Usage

```bash
$ npx package-create <project-name>
```

## Features

1. Linter / Formatter
  - Don't use
  - prettier
  - ESLint(Standard) - coming soon
2. Build Environment
  - Babel
  - TypeScript
3. Unit Testing
  - Don't use
  - Jest
  - AVA - coming soon
